## Template Usage Notes

- Every field should contain specific, extracted content — not placeholder language.
- If a section cannot be filled from the source document, note it as a gap: "[Not addressed in source document — requires clarification]".
- Keep the total output to one page when printed (roughly 500-700 words).
- Use bullet points over paragraphs for scannability.
- Include owners and dates wherever available.
